# comp_graph
Университетский курс по компьютерной графике. Задание на реализацию трассировки лучей в С++

Были реализованы:
- Локальное освещение по модели фонга,
- Тени
- Зеркальные отражения
- 3 различных материала
- 2 примитива (плоскость, шар)
